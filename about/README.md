# 关于页面配置

这个文件夹包含了关于页面的配置文件。

## 文件说明

### config.json

关于页面的配置文件，包含以下配置项：

#### title - 页面标题
- 类型: 字符串
- 说明: 关于页面的主标题
- 示例: "关于我"

#### subtitle - 页面副标题
- 类型: 字符串
- 说明: 关于页面的副标题
- 示例: "记录生活，分享技术，探索未知的世界"

#### introduction - 博客介绍
- 类型: 字符串数组
- 说明: 博客介绍段落，每个元素为一个段落
- 示例:
  ```json
  [
    "欢迎来到我的个人博客！这是一个使用 Astro 框架搭建的静态博客。",
    "博客支持深色模式、文章分类、标签系统等特性。"
  ]
  ```

#### techStack - 技术栈
- 类型: 对象数组
- 说明: 博客使用的技术栈列表
- 每个对象包含:
  - `name`: 技术名称
  - `desc`: 技术描述
- 示例:
  ```json
  [
    { "name": "Astro", "desc": "现代静态站点生成器" },
    { "name": "Markdown", "desc": "简洁的写作方式" }
  ]
  ```

#### features - 博客功能
- 类型: 字符串数组
- 说明: 博客支持的功能列表
- 示例:
  ```json
  [
    "深色/浅色主题切换",
    "文章分类和标签系统",
    "文章归档功能"
  ]
  ```

## 使用方法

1. 打开 `config.json` 文件
2. 修改相应的配置项
3. 保存文件
4. 重新构建博客（`npm run build` 或 `npm run dev`）

## 注意事项

- 请确保JSON格式正确
- introduction、techStack、features都是数组，可以添加或删除元素
- 修改配置后需要重新构建博客才能看到效果
